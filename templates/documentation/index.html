Overview

Write docs that feel clear, navigable, and substantial.

This starter gives you a polished reading experience with a sticky sidebar, article cards, consistent heading rhythm, and enough visual structure to support long-form technical content.

Use it for documentation, changelogs, onboarding portals, or internal playbooks.
Install

Drop the framework in and start writing.

The starter is intentionally simple: structure, navigation, and content blocks without unnecessary framework ceremony.

<link rel="stylesheet" href="vanduo.min.css"> <script src="vanduo.min.js"></script> <script>Vanduo.init();</script>
