Modal

Demo Modal

This is the Vanduo modal component. It supports backdrop click to close, keyboard escape, and the close button.

<!-- Trigger button -->
<button class="btn btn-primary" data-modal="#my-modal">Open Modal</button>

<!-- Modal structure -->
<div class="modal" id="my-modal">
  <div class="modal-backdrop"></div>
  <div class="modal-dialog">
    <div class="modal-content">
      <div class="modal-header">
        <h3 class="modal-title">Title</h3>
        <button class="modal-close" data-dismiss="modal">&times;</button>
      </div>
      <div class="modal-body">Content</div>
      <div class="modal-footer">
        <button class="btn btn-secondary" data-dismiss="modal">Close</button>
      </div>
    </div>
  </div>
</div>
// Open modal programmatically
const modal = Modal.open('#my-modal');

// Close modal
Modal.close('#my-modal');

// Listen for events
document.querySelector('#my-modal')
  .addEventListener('modal:open', (e) => {
    console.log('Modal opened');
  });

// Open item
item.classList.add('is-open');

API Reference

Tooltips

Attribute / Class Description Type
data-tooltip="[text]" Initializes a tooltip on the element with the specified text content. Attribute
data-tooltip-placement="[pos]" Sets placement (top, right, bottom, left). Defaults to top. Attribute

Dropdowns

Class Name Description Type
.vd-dropdown Base container class that establishes positioning context. Component
.vd-dropdown-toggle Button or link that toggles the dropdown menu visibility on click. Sub-component
.vd-dropdown-menu The hidden menu container that appears when activated. Sub-component
.vd-dropdown-item Individual interactive links or buttons within the popup menu. Sub-component
.vd-dropdown-divider Horizontal line used to separate groups of items. Sub-component

Modals

Class / Attribute Description Type
.vd-modal Outer wrapper for the modal dialog. Usually hidden by default. Component
.vd-modal-backdrop Semi-transparent overlay behind the modal dialog. Sub-component
.vd-modal-dialog Positioning container that centers the modal content. Layout
.vd-modal-content The actual visual card/box of the modal. Sub-component
.vd-modal-header Container for the modal title and close button. Sub-component
.vd-modal-body Main content area of the modal with default padding. Sub-component
.vd-modal-footer Bottom area typically used for action buttons. Sub-component
data-modal="#id" Attribute on a trigger button to open the target modal on click. Attribute
data-dismiss="modal" Attribute on elements inside the modal that should close it. Attribute

Accordions

Class Name Description Type
.accordion Base container class for a group of collapsible items. Component
.accordion-item Wrapper for a single header/body pair. Sub-component
.accordion-header Clickable button that toggles the visibility of the body. Sub-component
.accordion-icon Chevron or icon element that animates based on state. Sub-component
.accordion-body The collapsible area that hides or reveals its content. Sub-component
.accordion-content Inner wrapper providing padding to the body content. Sub-component
