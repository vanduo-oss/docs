Tabs

Basic Tabs

Content for Tab 1. This is the first tab panel.

Content for Tab 2. This is the second tab panel.

Content for Tab 3. This is the third tab panel.

<div class="vd-tabs">
  <div class="vd-tab-list" role="tablist">
    <button class="vd-tab-link is-active" data-tab="tab1">Tab 1</button>
    <button class="vd-tab-link" data-tab="tab2">Tab 2</button>
    <button class="vd-tab-link" data-tab="tab3">Tab 3</button>
  </div>
  <div class="vd-tab-content">
    <div class="vd-tab-pane is-active" id="tab1">
      <p>Content for Tab 1. This is the first tab panel.</p>
    </div>
    <div class="vd-tab-pane" id="tab2">
      <p>Content for Tab 2. This is the second tab panel.</p>
    </div>
    <div class="vd-tab-pane" id="tab3">
      <p>Content for Tab 3. This is the third tab panel.</p>
    </div>
  </div>
</div>

API Reference

Class / Attribute Description Type
.vd-tabs Base container for the entire tabs component. Component
.vd-tabs-boxed Modifier class to apply boxed tab styling. Modifier
.vd-tabs-pills Modifier class to apply pill tab styling. Modifier
.vd-tab-list Wrapper for all the individual tab links/buttons. Sub-component
.vd-tab-link Clickable button representing a single tab. Sub-component
.vd-tab-content Wrapper for all the tab panes/content areas. Sub-component
.vd-tab-pane Individual content area associated with a tab. Sub-component
.is-active Active state class for both .vd-tab-link and .vd-tab-pane. State
data-tab="[id]" Attribute on .vd-tab-link that targets a .vd-tab-pane's ID. Attribute
