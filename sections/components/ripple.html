Ripple / Waves

Basic Ripple on Buttons

<!-- Add .vd-ripple to any clickable element -->
<button class="vd-btn vd-btn-primary vd-ripple">
  Primary Ripple
</button>

<!-- Or use the data attribute -->
<button class="vd-btn vd-btn-primary" data-vd-ripple>
  Primary Ripple
</button>

Color Variants

<!-- Default light ripple -->
<button class="vd-btn vd-ripple">Default</button>

<!-- Dark ripple (for light backgrounds) -->
<button class="vd-btn vd-ripple vd-ripple-dark">Dark</button>

<!-- Primary-coloured ripple -->
<button class="vd-btn vd-ripple vd-ripple-primary">Primary</button>

<!-- Light ripple (for dark backgrounds) -->
<button class="vd-btn vd-ripple vd-ripple-light">Light</button>

Ripple on Non-button Elements

Click anywhere on this div

Ripple works on any element — not just buttons

<!-- Ripple on a generic div -->
<div class="vd-ripple" data-vd-ripple
     style="position:relative; overflow:hidden;">
  Click anywhere on this div
</div>

Data Attribute Variants

<!-- Variant via data-vd-ripple value -->
<button data-vd-ripple>Default</button>
<button data-vd-ripple="dark">Dark</button>
<button data-vd-ripple="primary">Primary</button>

API Reference

CSS Classes

Class Description Type
.vd-ripple Enable ripple effect on an element (requires position:relative; overflow:hidden on non-button elements) Base
.vd-ripple-dark Dark ripple wave — best on light/white backgrounds Modifier
.vd-ripple-primary Ripple wave in the theme's primary colour Modifier
.vd-ripple-light Light/white ripple wave — best on dark backgrounds Modifier
.vd-ripple-wave Internal class applied to the expanding circle element (auto-generated by JS) Internal

Data Attributes

Attribute Description Default
data-vd-ripple Enable ripple via data attribute. Empty value uses default light wave. light
data-vd-ripple="dark" Dark ripple variant via data attribute —
data-vd-ripple="primary" Primary-colour ripple variant via data attribute —

JavaScript Methods

Method Description
VanduoRipple.init() Initialize ripple listeners on all .vd-ripple and [data-vd-ripple] elements
VanduoRipple.destroy(el) Remove ripple event listeners from the given element

Accessibility

  • The ripple is a purely visual effect and does not affect DOM focus or screen reader output
  • The generated .vd-ripple-wave element uses aria-hidden="true"
  • Ripple animation respects prefers-reduced-motion: reduce — the wave is instant rather than animated
  • Non-button elements with ripple should include appropriate role and tabindex for keyboard access
  • Colour contrast of the ripple wave meets WCAG guidelines against all standard button backgrounds
