Pagination (JavaScript)

Add data-pagination on the list; set data-total-pages, data-current-page, and optional data-max-visible. The component renders controls and emits pagination:change when the active page updates.

    <!-- Vanduo initializes elements with data-pagination on Vanduo.init() -->
<nav aria-label="Page navigation">
  <ul class="vd-pagination"
    data-pagination
    data-total-pages="12"
    data-current-page="3"
    data-max-visible="7">
  </ul>
</nav>

    API Reference

    Class Name Description Type
    .vd-breadcrumbs Wrapper for breadcrumb navigation. Layout
    .vd-breadcrumb-item Individual breadcrumb list item. Add .active for the current page. Component
    .vd-pagination Wrapper for pagination controls. Layout
    .vd-pagination-item Individual pagination list item. Add .active for the current page. Component
    .vd-pagination-link Clickable link inside a pagination item. Element
    data-pagination On the ul.vd-pagination root: Vanduo initializes dynamic page controls (Vanduo.init()). JavaScript API
    data-total-pages Total number of pages (default 1). JavaScript API
    data-current-page Active page index (default 1). JavaScript API
    data-max-visible Max page numbers shown before ellipsis (default 7). JavaScript API
    data-demo-no-nav Docs SPA helper: preventDefault on placeholder <a href="#"> clicks so hash routing stays put. Documentation
