Card Glow Effect

Add a gentle glow effect to any card with the .vd-card-glow class. On hover, cards emit a soft primary-colored glow using CSS box-shadow. 

<!-- Card with glow effect on hover -->
<div class="vd-card vd-card-glow">
  <div class="vd-card-body">Hover over this card</div>
</div>

Tip: The glow color adapts to your theme's primary color. Works beautifully with dark mode too!

<div class="vd-card">
    <div class="vd-card-header">
    <h3>Card Title</h3>
  </div>
    <div class="vd-card-body">
    <p>Card content goes here.</p>
  </div>
    <div class="vd-card-footer">
        <button class="vd-btn vd-btn-primary vd-btn-sm">Action</button>
  </div>
</div>

<!-- Outlined variant -->
<div class="vd-card vd-card-outlined">...</div>

<!-- Filled variant -->
<div class="vd-card vd-card-filled">...</div>
.vd-card {
  background-color: var(--card-bg);
  border-radius: var(--card-border-radius);
  box-shadow: var(--shadow-md);
  overflow: hidden;
}

.vd-card-outlined {
  box-shadow: none;
  border: 1px solid var(--border-color);
}

.vd-card-filled {
  box-shadow: none;
  background-color: var(--bg-secondary);
}

Class Reference

Class Description Type
.vd-card Base card container — required on all cards Base
.vd-card-outlined Outlined card with border, no shadow Variant
.vd-card-filled Filled card with secondary background, no border Variant
.vd-card-glow Adds soft primary-colored glow on hover Modifier
.vd-card-actionable Enables hover lift and interactive glow effect Modifier
.vd-card-header Card header section with bottom border Structure
.vd-card-body Card body section with padding Structure
.vd-card-footer Card footer section with top border Structure
