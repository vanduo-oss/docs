API Reference

CSS Classes

Class Description
.vd-timeline Base container. Draws a vertical line and positions items along it.
.vd-timeline-item Individual event wrapper containing a marker and content block.
.vd-timeline-marker Circular dot positioned on the timeline track. Default size is 12px.
.vd-timeline-marker-lg Larger marker (24px) that can contain an icon.
.vd-timeline-content Text block beside the marker holding title, date, and description.
.vd-timeline-title Heading text for the event.
.vd-timeline-date Small muted text for the event date or timestamp.
.vd-timeline-text Body text describing the event in detail.
.vd-timeline-alternating Modifier that alternates items left and right of the center line (desktop only).
.vd-timeline-success Colors the marker with the success (green) palette.
.vd-timeline-warning Colors the marker with the warning (amber) palette.
.vd-timeline-danger Colors the marker with the danger (red) palette.
No JavaScript needed. The timeline is a pure CSS component. No initialization or event handling is required.

Accessibility

  • Use an <ol> or add role="list" on the container and role="listitem" on each .vd-timeline-item so events are announced as an ordered sequence.
  • Markers are decorative — apply aria-hidden="true" on .vd-timeline-marker icons.
  • Ensure .vd-timeline-date text uses a <time> element with a datetime attribute for machine-readable dates.
  • Color variants must not be the sole indicator of state — pair them with text labels or icons.
  • The alternating layout collapses to a single column on narrow viewports, maintaining natural reading order.
