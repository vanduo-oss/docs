Progress

Linear progress bars with semantic color variants, multiple sizes, striped and animated styles, and indeterminate mode. Progress bars are fully CSS-driven — set the width via inline style or utility classes and the bar animates smoothly.

Accessibility

  • Wrap progress bars in an element with role="progressbar", aria-valuenow, aria-valuemin="0", and aria-valuemax="100".
  • Provide an aria-label or aria-labelledby describing what is progressing (e.g., "Uploading file.jpg").
  • For indeterminate bars, omit aria-valuenow and add aria-valuetext="Loading..." instead.
  • Animations respect prefers-reduced-motion: reduce — stripes and indeterminate motion are disabled.
  • Dark mode automatically adjusts --vd-progress-bg for sufficient contrast.
