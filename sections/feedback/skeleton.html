Usage

<!-- Text Skeleton -->
<div class="vd-skeleton vd-skeleton-text" style="width: 100%;"></div>
<div class="vd-skeleton vd-skeleton-text" style="width: 80%;"></div>

<!-- Circle Skeleton (avatars) -->
<div class="vd-skeleton vd-skeleton-circle"></div>

<!-- Rectangle Skeleton (images, cards) -->
<div class="vd-skeleton vd-skeleton-rect" style="width: 100%; height: 100px;"></div>

<!-- Card Skeleton Pattern -->
<div class="vd-d-flex vd-gap-6">
  <div class="vd-skeleton vd-skeleton-circle"></div>
  <div style="flex: 1;">
    <div class="vd-skeleton vd-skeleton-text" style="width: 60%;"></div>
    <div class="vd-skeleton vd-skeleton-text" style="width: 40%;"></div>
  </div>
</div>

API Reference

Class Name Description Type
.vd-skeleton Base component class required for all Vanduo skeleton loaders. Applies the shimmer animation. Component
.vd-skeleton-text Shape modifier for representing lines of text. Features a small border radius. Modifier
.vd-skeleton-circle Shape modifier for representing avatars or circular icons. Applies full border radius. Modifier
.vd-skeleton-rect Shape modifier for representing images, cards, or larger blocks. Sharp corners default. Modifier
