Features

Zero Dependencies

Pure HTML, CSS, and JavaScript. No mandatory build tools, no complex setup required, no friction.

Fibonacci Design

Spacing, typography, and grid layouts built on the Golden Ratio and Fibonacci sequence for natural harmony.

Modular CSS

Import the full framework or pick only the components you need. No bloat, just what you use.

Accessibility First

ARIA attributes, focus-visible states, reduced motion support, and semantic HTML built-in.

Dark Mode Ready

Light, dark, and system themes with smooth transitions. Includes a customizable theme switcher component.

Lightweight & Fast

Minimal footprint with optimized CSS. No JavaScript required for styling—scripts are optional enhancements.

Phosphor Icons

1,500+ icons in 6 weights, bundled locally. No CDN required.

What's New in v1.2.3

  • Organization Transfer — Vanduo has officially moved to its new home at the vanduo-oss GitHub organization, paving the way for community scaling and better collaboration.
  • Scoped NPM Package — The framework is now published as a scoped package: @vanduo-oss/framework, preventing naming collisions and clearly signaling official packages.
  • Package Deprecation — The legacy vanduo-framework package has been formally deprecated. All downstream projects must update their package.json and import statements.
  • Docs Synced to v1.2.3 — All CDN tags, Quick Start commands, and integration guides have been aggressively audited and migrated to the new scope.
