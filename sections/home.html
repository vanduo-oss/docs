Zero Dependencies

Ship pure HTML, CSS, and JavaScript straight to the browser. No npm install, no bundler, no build step — drop the CDN link in and you're productive in under a minute.

Fibonacci Design

Tap to learn more

Fibonacci Design

Every spacing unit, font-size step, and grid column is derived from the Golden Ratio and the Fibonacci sequence, so layouts feel balanced without manual tweaking.

Modular CSS

Tap to learn more

Modular CSS

Import the full vanduo.css or cherry-pick individual component files. Tree-shake what you don't need and keep your payload lean.

Accessibility First

Tap to learn more

Accessibility First

Built-in ARIA roles, visible focus rings, semantic HTML, keyboard-first interactions, and prefers-reduced-motion support — accessibility is on by default, not an add-on.

Dark Mode Ready

Tap to learn more

Dark Mode Ready

Light, dark, and system-synced themes with smooth CSS-variable transitions. Ships with a drop-in theme switcher and a live customizer.

Lightweight & Fast

Tap to learn more

Lightweight & Fast

A minimal, cache-friendly CSS footprint. Styling works with JavaScript disabled — scripts are progressive enhancements, never a requirement.

Pure & Simple

Tap to learn more

Pure & Simple

Readable, framework-free source. No JSX, no transpilation, no magic — just standards-compliant HTML, CSS, and vanilla JS you can audit line by line.

Conflict-Free

Tap to learn more

Conflict-Free

Every class is namespaced with the vd- prefix so Vanduo can safely coexist with Bootstrap, Tailwind, or your own legacy styles without cascade surprises.

Developer First

Developer First

Consistent naming, predictable DOM, fully documented components, and copy-paste examples. Easy to extend, theme, and contribute back to.
