Typography

Font Families

Vanduo ships with 5 curated font options for runtime switching. These are the same options exposed by FontSwitcher and ThemeCustomizer in the framework and docs.

Supported Font Keys

  • ubuntu — Default humanist sans-serif option
  • open-sans — Neutral, highly readable sans-serif
  • lato — Friendly, rounded sans-serif
  • jetbrains-mono — Monospace optimized for code
  • system — Use platform native system stack

Behavior Notes

  • Default preference is ubuntu.
  • Setting data-font="system" is equivalent to removing data-font.
  • Preference persists in localStorage.vanduo-font-preference.
  • Legacy keys (for example inter, rubik) are normalized to supported options in docs runtime glue.

Switching Fonts

Use the data-font attribute on the HTML element:

<!-- Set font on HTML element -->
<html data-font="ubuntu">

<!-- Or use the Theme Customizer (paint roller icon) to preview and select fonts -->
<button data-theme-customizer-trigger>
    <i class="ph ph-paint-roller"></i>
</button>

Font preference is persisted to localStorage and restored on page load.

Adding More Fonts

Need a different family than the built-in curated set? You have two options:

Option 1: Self-Host (Recommended)

Download fonts from Google Fonts and add them to your fonts/ folder. Follow the same pattern as existing fonts: 

1. Create folder: fonts/my-font/
2. Add woff2 files
3. Add @font-face rules to css/core/fonts.css
4. Add [data-font="my-font"] selector to css/core/typography.css
5. Register in js/components/font-switcher.js and js/components/theme-customizer.js
6. (Docs site) add key mapping in docs/js/app.js docsFontOptions/SUPPORTED_CUSTOMIZER_FONTS
Option 2: Google Fonts CDN

For quick prototyping, use a Google Fonts CDN link in your HTML head:

<link rel="preconnect" href="https://fonts.googleapis.com">
<link href="https://fonts.googleapis.com/css2?family=Roboto:wght@400;500;700&display=swap" rel="stylesheet">

<style>
  [data-font="roboto"] {
    --font-family-sans: 'Roboto', sans-serif;
  }
</style>

API Reference

Class Name / Attribute Description Type
data-font="[name]" Applies a supported font stack on <html> (e.g., lato, open-sans, ubuntu, jetbrains-mono, system). Attribute
.vd-text-[variant] Modifies text color (e.g., -primary, -success, -error, -muted, -white). Utility
.vd-bg-[variant] Modifies background color (e.g., -primary, -success, -error, -dark, -light). Utility
.vd-text-[size] Modifies text sizing (e.g., -sm for small, -lg for large). Utility
.vd-font-bold Applies a bold font weight. Utility
.vd-text-italic Applies italic font style. Utility
