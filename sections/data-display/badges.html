API Reference

Class Name Description Type
.vd-badge Base component class required for all Vanduo badges. Component
.vd-badge-[variant] Color modifier class (e.g., -primary, -success, -warning, -error, -info). Modifier
.vd-badge-[size] Size modifier class (e.g., -sm for small, -lg for large). Defaults to medium if omitted. Modifier
.vd-badge-pill Shape modifier that applies fully rounded corners (pill shape). Modifier
