Utilities Cheat Sheet Guide

Vanduo includes a comprehensive set of utility classes based on the Fibonacci scale (8, 13, 21, 34, 55...). This cheat sheet provides a quick reference for all available utilities.

Spacing (Margin & Padding)

Format: vd-{property}-{direction}-{size}

Property Direction(s) Sizes (px)
m (margin)
p (padding)		 none (all sides)
t (top), b (bottom)
l (left), r (right)
x (left & right)
y (top & bottom) 		0 (0px)
1 (8px)
2 (13px)
3 (21px)
4 (34px)
5 (55px)
6 (89px)
7 (144px)
8 (233px)
Examples: vd-mb-4 = Margin Bottom 34px
vd-px-3 = Padding X (Left/Right) 21px
vd-m-0 = Margin 0px on all sides 
<div class="vd-card vd-mb-5">
  <div class="vd-card-body vd-p-4">
    <h3 class="vd-mb-3">Card Title</h3>
    <p class="vd-mb-0">Content with controlled spacing.</p>
  </div>
</div>

<div class="vd-py-8 vd-px-5">
  <!-- Section with generous vertical padding -->
</div>

Display & Flexbox

Class Description
vd-d-nonedisplay: none;
vd-d-blockdisplay: block;
vd-d-inline-blockdisplay: inline-block;
vd-d-flexdisplay: flex;
vd-d-inline-flexdisplay: inline-flex;
vd-flex-colflex-direction: column;
justify-startjustify-content: flex-start;
justify-centerjustify-content: center;
justify-betweenjustify-content: space-between;
items-centeralign-items: center;
 
<!-- Centered flex row with space between -->
<div class="vd-d-flex justify-between items-center">
  <span>Left content</span>
  <span>Right content</span>
</div>

<!-- Vertical stack -->
<div class="vd-d-flex vd-flex-col items-center">
  <h3>Title</h3>
  <p>Subtitle</p>
  <button class="vd-btn vd-btn-primary">Action</button>
</div>

<!-- Hide on mobile, show on desktop -->
<div class="vd-d-none vd-d-md-block">
  Desktop-only sidebar
</div>
