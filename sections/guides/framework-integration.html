Framework Integration Guide

Vanduo remains framework-agnostic in v1.4.1, but the recommended JavaScript pattern is now scoped: initialize the DOM subtree your app just mounted, and destroy that same subtree before your framework removes it.

The Pattern
  1. Import vanduo.css once in your global app entry.
  2. After a route, view, or component subtree mounts, call Vanduo.init(root).
  3. Before that subtree unmounts, call Vanduo.destroy(root).
Vanduo.init(root);              // initialize only inside root
Vanduo.destroy(root);           // clean up only inside root
Vanduo.reinit('dropdown', root); // reset one component inside root
Vanduo.getComponent('lazyLoad'); // canonical lowerCamelCase registry name

Omit root only for full-document pages or one-time static bootstraps.

React
import { useEffect, useRef } from 'react';
import { Vanduo } from '@vanduo-oss/framework';
import '@vanduo-oss/framework/dist/vanduo.css';

export function VanduoBoundary({ children }) {
  const rootRef = useRef(null);

  useEffect(() => {
    const root = rootRef.current;
    Vanduo.init(root);
    return () => Vanduo.destroy(root);
  }, []);

  return <div ref={rootRef}>{children}</div>;
}
Vue
<script setup>
import { onMounted, onUnmounted, ref } from 'vue';
import { Vanduo } from '@vanduo-oss/framework';

const root = ref(null);

onMounted(() => Vanduo.init(root.value));
onUnmounted(() => Vanduo.destroy(root.value));
</script>

<template>
  <div ref="root"><slot /></div>
</template>
Svelte
<script>
  import { onMount } from 'svelte';
  import { Vanduo } from '@vanduo-oss/framework';

  let root;

  onMount(() => {
    Vanduo.init(root);
    return () => Vanduo.destroy(root);
  });
</script>

<div bind:this={root}><slot /></div>
Dynamic DOM

When you replace a subtree yourself, destroy the old subtree before assigning new HTML, then initialize the new subtree.

async function replacePanel(panel, url) {
  Vanduo.destroy(panel);
  panel.innerHTML = await fetch(url).then((res) => res.text());
  Vanduo.init(panel);
}

For one component family, use Vanduo.reinit('tabs', panel) or another canonical lowerCamelCase registry name.
