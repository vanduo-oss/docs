Music Player

A fully accessible HTML5 audio player component with transport controls, volume, and optional shuffle, seek bar, and playlist features. Zero external dependencies, powered by the native Audio API, and it automatically adapts to the active Vanduo theme.

Minimal Player

Default config — play/pause, prev/next, volume, track name

<div id="my-player" class="vd-music-player"></div>
const tracks = [
  { name: 'Track 01', url: './music/track-01.mp3' },
  { name: 'Track 02', url: './music/track-02.mp3' },
];

VanduoMusicPlayer.initPlayer(
  document.getElementById('my-player'),
  { tracks }
);

With Progress Bar

Enable showProgress: true for a seek bar + elapsed/duration times

VanduoMusicPlayer.initPlayer(el, {
  tracks,
  showProgress: true,
});

With Shuffle

Start shuffled with shuffle: true, or tap the shuffle button to toggle

VanduoMusicPlayer.initPlayer(el, {
  tracks,
  shuffle: true,
});

With Playlist

Enable showPlaylist: true to reveal a collapsible track list

VanduoMusicPlayer.initPlayer(el, {
  tracks,
  showPlaylist: true,
});

Programmatic API

const el = document.getElementById('my-player');

// Transport
VanduoMusicPlayer.play(el);
VanduoMusicPlayer.pause(el);
VanduoMusicPlayer.toggle(el);
VanduoMusicPlayer.next(el);
VanduoMusicPlayer.previous(el);

// Go to a specific track (0-based index)
VanduoMusicPlayer.setTrack(el, 2);

// Volume (0 to 1)
VanduoMusicPlayer.setVolume(el, 0.75);

// Toggle shuffle programmatically
VanduoMusicPlayer.shuffle(el);

// Get current state
const state = VanduoMusicPlayer.getState(el);
// { isPlaying, currentIndex, currentTrack, volume, shuffle, tracks }

// Tear down
VanduoMusicPlayer.destroy(el);
VanduoMusicPlayer.destroyAll();

// Re-init after dynamic markup changes
Vanduo.reinit('musicPlayer');

Custom Events

el.addEventListener('musicplayer:play', () => {
    console.log('Playback started');
});

el.addEventListener('musicplayer:pause', () => {
    console.log('Playback paused');
});

el.addEventListener('musicplayer:trackchange', (event) => {
    const { index, name, url } = event.detail;
    console.log('Track changed to:', index, name, url);
});

el.addEventListener('musicplayer:volumechange', (event) => {
    console.log('Volume:', event.detail.volume);
});

el.addEventListener('musicplayer:ended', () => {
    // Fires only when autoAdvance is false on the last track.
    console.log('Playback ended');
});

Options Reference

Option Type Default Description
tracks Array [] Array of { name, url } objects. name is displayed; url is the audio file path.
volume number 0.5 Initial playback volume between 0 and 1.
shuffle boolean false Shuffle the track list on init. Also shows the shuffle toggle button.
showProgress boolean false Show a seek bar with elapsed/total time display.
showPlaylist boolean false Show a collapsible track list panel. Also shows the shuffle button.
autoAdvance boolean true Automatically play the next track when the current one ends.
