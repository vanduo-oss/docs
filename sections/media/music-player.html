Music Player

A fully accessible HTML5 audio player component with transport controls, volume, optional shuffle, seek bar, playlist, glass surface, and a detachable floating mode (corner placement, pointer-drag reposition, minimizable). Zero external dependencies, powered by the native Audio API, and it automatically adapts to the active Vanduo theme.

Install the player separately from the core framework. All live demos below use a bundled sample playlist from Invent the Universe, with Pale Blue Dot as the default first track.

Install

pnpm add @vanduo-oss/music-player

Load vanduo-music-player.css and vanduo-music-player.iife.js after the core Vanduo bundle, then call Vanduo.init(). The player registers as musicPlayer and auto-inits data-music-player elements.

import VanduoMusicPlayer from '@vanduo-oss/music-player';
import '@vanduo-oss/music-player/css';

VanduoMusicPlayer.initPlayer(el, { tracks });

With Progress Bar

Enable showProgress: true for a seek bar + elapsed/duration times

VanduoMusicPlayer.initPlayer(el, {
  tracks,
  showProgress: true,
});
With Playlist + Progress

Enable showPlaylist: true and showProgress: true for a fuller player. Shuffle is always available from the control bar — use shuffle: true only to start in shuffled order.

VanduoMusicPlayer.initPlayer(el, {
  tracks,
  showPlaylist: true,
  showProgress: true,
});

Glass surface

Add glass: true or the vd-music-player-glass class to any player size or layout. Uses the same frosted tokens as vd-glass in the core framework.

VanduoMusicPlayer.initPlayer(el, {
  tracks,
  showProgress: true,
  glass: true,
});
Detachable — fixed corners

detachable: true, minimizable: true, draggable: false. Snap to bottom-left, bottom-right, top-left, or top-right via floatingPosition or detach(el, position).

Detachable — draggable

draggable: true adds a drag handle when floating. Still supports minimize/expand. Free positioning overrides corner presets.

VanduoMusicPlayer.initPlayer(el, {
  tracks,
  showProgress: true,
  showPlaylist: true,
  detachable: true,
  draggable: true,
  minimizable: true,
});

VanduoMusicPlayer.detach(el, 'bottom-right');
VanduoMusicPlayer.minimize(el);

Programmatic API
const el = document.getElementById('my-player');

// Transport
VanduoMusicPlayer.play(el);
VanduoMusicPlayer.pause(el);
VanduoMusicPlayer.toggle(el);
VanduoMusicPlayer.next(el);
VanduoMusicPlayer.previous(el);

// Go to a specific track (0-based index)
VanduoMusicPlayer.setTrack(el, 0);

// Volume (0 to 1)
VanduoMusicPlayer.setVolume(el, 0.75);

// Toggle shuffle programmatically
VanduoMusicPlayer.shuffle(el);

// Float above the page (requires detachable: true at init)
VanduoMusicPlayer.detach(el, 'bottom-right');
VanduoMusicPlayer.attach(el);
VanduoMusicPlayer.setPosition(el, 'top-right');
VanduoMusicPlayer.setPosition(el, { x: 24, y: 120 });

// Minimize / expand (requires minimizable: true at init)
VanduoMusicPlayer.minimize(el);
VanduoMusicPlayer.expand(el);
VanduoMusicPlayer.toggleMinimize(el);

// Get current state
const state = VanduoMusicPlayer.getState(el);
// { isPlaying, currentIndex, currentTrack, volume, shuffle, tracks, isDetached, isMinimized }

// Tear down
VanduoMusicPlayer.destroy(el);
VanduoMusicPlayer.destroyAll();

// Re-init after dynamic markup changes
Vanduo.reinit('musicPlayer');
Custom Events
el.addEventListener('musicplayer:play', () => {
    console.log('Playback started');
});

el.addEventListener('musicplayer:pause', () => {
    console.log('Playback paused');
});

el.addEventListener('musicplayer:trackchange', (event) => {
    const { index, name, url } = event.detail;
    console.log('Track changed to:', index, name, url);
});

el.addEventListener('musicplayer:volumechange', (event) => {
    console.log('Volume:', event.detail.volume);
});

el.addEventListener('musicplayer:ended', () => {
    // Fires only when autoAdvance is false on the last track.
    console.log('Playback ended');
});

el.addEventListener('musicplayer:detach', () => {
    console.log('Player is floating');
});

el.addEventListener('musicplayer:attach', () => {
    console.log('Player was docked');
});

el.addEventListener('musicplayer:minimize', () => {
    console.log('Minimized');
});

el.addEventListener('musicplayer:expand', () => {
    console.log('Expanded');
});

Options Reference
Option Type Default Description
tracks Array [] Array of { name, url } objects. name is displayed; url is the audio file path.
volume number 0.5 Initial playback volume between 0 and 1.
shuffle boolean false Shuffle the track list on init. Also shows the shuffle toggle button.
showProgress boolean false Show a seek bar with elapsed/total time display.
showPlaylist boolean false Show a collapsible track list panel. Also shows the shuffle button.
autoAdvance boolean true Automatically play the next track when the current one ends.
glass boolean false Frosted-glass surface (vd-music-player-glass); pairs with theme --vd-glass-* tokens.
detachable boolean false Show detach/attach controls and allow VanduoMusicPlayer.detach() to float the player above the page.
floatingPosition string 'bottom-right' After detaching: 'bottom-left', 'bottom-right', 'top-left', or 'top-right' (override via detach(el, position)).
draggable boolean false When detachable is true, show a drag handle and allow pointer-dragging while floating (not HTML5 list DnD).
minimizable boolean false Show a control to collapse the player to essential transport + volume.
startMinimized boolean false On the first detach(), start in the minimized layout.
persistPosition boolean false Save floating {x, y} to localStorage (via safeStorage when available).
persistKey string '' (from element id) Storage key suffix for persistPosition; defaults from the element’s id attribute.
Accessibility
  • The container is a role="region" labeled Music Player; transport controls sit in a role="group" labeled Playback controls.
  • Buttons expose aria-label text (Play/Pause, Previous/Next track, Volume, Seek, playlist toggle with aria-expanded, shuffle with aria-pressed).
  • The current track name uses aria-live="polite" and aria-atomic="true" so screen readers announce track changes.
  • Playlist items set aria-current on the active row; decorative icons use aria-hidden="true".
  • Play/Pause supports Space and Enter on the play button; range inputs are keyboard-focusable with visible :focus-visible outlines.
  • Floating toolbar controls (detach, attach, minimize, drag handle) include descriptive aria-label values; minimize toggles aria-expanded.
  • Glass and floating transitions respect prefers-reduced-transparency and prefers-reduced-motion in the component stylesheet.
