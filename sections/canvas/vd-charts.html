Vanduo Charts

Vanduo Charts is a standalone SVG-first package for common dashboard charts. It is installed separately from the framework, renders accessible SVG, reads Vanduo theme tokens, and can auto-initialize through the v1.4 lifecycle system when the browser bundle is loaded.

Install

pnpm add @vanduo-oss/charts

JavaScript API

import { DonutChart } from '@vanduo-oss/charts';
import '@vanduo-oss/charts/css';

const chart = DonutChart({
  target: '#revenue-mix',
  data,
  label: 'channel',
  value: 'revenue',
  title: 'Revenue mix',
  tooltip: d => `${d.channel}: ${d.revenue}`,
  onSliceClick: ({ datum }) => console.log(datum)
});

chart.update({ data: nextData });
chart.resize();
chart.destroy();

Vanduo Auto Init

The browser bundle exposes window.VanduoCharts. When window.Vanduo is present, it registers as charts, so scoped Vanduo.init(root), Vanduo.destroy(root), and Vanduo.reinit('charts', root) work without framework changes.

<div
  data-vd-chart="donut"
  data-vd-label="channel"
  data-vd-value="revenue"
  data-vd-title="Revenue mix">

  { "channel": "Services", "revenue": 180 }

API Reference

Export Purpose
BarChartCategory bars with band scale, left/bottom axes, grid lines, tooltips, and bar click events.
LineChartLine path with optional points over category, number, or date-like x values.
AreaChartFilled area plus line path, sharing the same cartesian infrastructure as line charts.
ScatterChartCircle marks over numeric, date-like, or categorical x values.
DonutChartArc slices with default innerRadiusRatio: 0.62, center total, legend, tooltips, and slice click events.
PieChartSame polar renderer as DonutChart with innerRadiusRatio: 0.
Shared Options
Option Type Description
targetElement | selectorRequired mount element.
dataArrayPlain objects; null x/y/value rows are skipped.
x, ystring | functionCartesian accessors.
label, valuestring | functionPie and donut accessors.
title, descriptionstringRendered into SVG accessibility metadata.
tooltipfunction | string | falseCustom tooltip content or disable tooltips.
themeobjectOverride token-derived colors, text, grid, and axis values.

Accessibility

  • Each chart renders an SVG with role="img" and title/description metadata when provided.
  • Interactive marks are keyboard-focusable and respond to Enter or Space.
  • Tooltips use a polite status region and do not own data loading or dashboard state.
