Flowchart

Vanduo Flowchart is a standalone SVG editor for flow diagrams with zoom, canvas drag, draggable primitives, double-click text editing, a one-shot Arrow tool, selected-only connection handles, selected-edge reconnect handles, calmer side-aware snapping, per-edge route and line-weight controls, larger marker caps, keyboard copy-paste, selected-node resizing, junction primitives for branching, and a built-in JSON import/export panel. It lives outside the framework package and follows the same standalone package pattern as Vanduo Charts and Hex Grid.

Install

pnpm add @vanduo-oss/flowchart

JavaScript API

import { VdFlowchart } from '@vanduo-oss/flowchart';
import '@vanduo-oss/flowchart/css';

const editor = new VdFlowchart({
  element: document.getElementById('flowchart-demo'),
  data: {
    nodes: [
      { id: 'start', type: 'circle', x: 80, y: 80, text: 'Start' },
      { id: 'review', type: 'rounded-rect', x: 300, y: 92, text: 'Review request' }
    ],
    edges: [
      {
        id: 'edge-1',
        from: { nodeId: 'start', port: 'right' },
        to: { nodeId: 'review', port: 'left' },
        kind: 'arrow',
        route: 'orthogonal',
        strokeWidth: 3.5
      }
    ]
  }
});

editor.addNode({ type: 'diamond', text: 'Approved?' });
editor.addNode({ type: 'junction' });
editor.fitView();

const snapshot = editor.toJSON();
editor.load(snapshot);

Vanduo Auto Init

The browser bundle exposes window.VanduoFlowchart. When window.Vanduo is present, it registers as flowchart, so scoped Vanduo.init(root), Vanduo.destroy(root), and Vanduo.reinit('flowchart', root) work without framework changes.

API Reference

Method Purpose
addNode(node)Add a primitive near the current viewport center.
updateNode(id, patch)Change node type, text, size, or coordinates.
removeNode(id)Delete a node and its connected edges.
addEdge(edge)Create a line or arrow between two ports. Set route to curve, straight, or orthogonal, adjust strokeWidth for line weight, and use marker caps as needed; pointer-created edges snap to the nearest side of the target shape.
removeEdge(id)Delete a connection.
zoomIn(), zoomOut(), fitView(), resetView()Adjust viewport scale and framing.
startTextEdit(nodeId), stopTextEdit({ commit })Open or close the inline text editor used by double-click editing.
toJSON(), load(data), clear()Export, import, or reset the full document.
on(event, callback), off(event, callback)Subscribe to change, select, viewport, and connect. Resize completion emits change with reason: 'node:resize'.

Accessibility

  • The editor surface exposes a single labeled SVG region and keeps editor controls in semantic HTML.
  • Nodes and edges remain keyboard-deletable once the canvas is focused.
  • Connection handles stay out of hit-testing until a node is selected, the Arrow tool is armed, or a reconnect is in progress.
  • The JSON panel provides a direct text representation for import/export and testing workflows.
